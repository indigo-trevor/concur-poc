# Concur POC
Web application built for Concur.

## Environment
* [Slick](https://github.com/kenwheeler/slick)
* [Greensock](https://greensock.com/)
* [jQuery](https://jquery.com/)
* [SASS](https://github.com/sass/sass)
* [Gulp](https://gulpjs.com/
  )
## Development
```bash
$ npm install
```
Then run:

```bash
$ gulp start
```
